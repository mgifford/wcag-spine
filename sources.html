⚙️ Sync Workflow

Data is refreshed automatically every day via a GitHub Actions workflow. The sync script (scripts/sync_data.py) fetches from each upstream source, merges the results, and writes data/master_spine.json.

  1. Fetch latest axe-core and Alfa version numbers from npm registry
  2. Fetch ACT rules from wcag-mapping.json (W3C GitHub)
  3. Supplement with secondary-SC coverage from testcases.json
  4. Fetch Alfa SC mappings from alfa-rules/src/index.ts
  5. Enrich Alfa mappings from the Alfa EARL implementation report
  6. Fetch axe-core rule–SC mappings (GitHub API → rule JSON files; fallback to axe.js regex)
  7. Fetch ARRM role data (arrm-wcag-sc.csv) and structured tasks (arrm-all-tasks.csv)
  8. Fetch Section 508 FPC mapping (mapping-wcag-to-fpc.csv)
  9. Merge all sources into the spine; propagate ACT implementation data
  10. Write data/master_spine.json with updated meta.generated date
  11. Deploy to GitHub Pages
